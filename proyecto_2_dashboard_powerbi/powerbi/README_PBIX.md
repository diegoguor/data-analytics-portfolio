# Archivo .pbix

En este entorno no se genera automáticamente un `.pbix` real porque requiere Power BI Desktop.

Este proyecto incluye:
- Dataset listo (`data/`)
- SQL de extracción (`sql_queries/`)
- Medidas DAX y guía de dashboard (`documentacion_metricas.md` + `instrucciones_dashboard_powerbi.md`)

Con estos archivos puedes construir el .pbix en 30-60 minutos siguiendo la guía paso a paso.
