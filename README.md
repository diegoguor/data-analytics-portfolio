Bienvenido/a a mi portafolio profesional. Aquí encontrarás proyectos end-to-end orientados a **análisis de datos, inteligencia de negocio y analítica predictiva**, con enfoque en decisiones de negocio basadas en datos.

## 👨‍💼 Sobre mí
Soy **Analista de Datos / BI** con experiencia en:
- Construcción de pipelines reproducibles y análisis exploratorio.
- Diseño de métricas de negocio y tableros ejecutivos.
- Aplicación de modelos de machine learning para casos reales (como churn).

Me enfoco en convertir datos en **insights accionables** y comunicar resultados de forma clara para equipos técnicos y de negocio.

## 📂 Proyectos destacados

### 1) 📊 [Análisis de Ventas con Python + SQL](./proyecto_1_analisis_ventas_python)
**Objetivo:** entender el rendimiento comercial e-commerce mediante EDA, KPIs y consultas SQL.

**Incluye:**
- Generación y análisis de dataset de ventas.
- KPIs ejecutivos (ventas, ticket, margen, devoluciones).
- Visualizaciones y reporte SQL en Markdown.

### 2) 📈 [Dashboard Interactivo en Power BI](./proyecto_2_dashboard_powerbi)
**Objetivo:** construir un dashboard de desempeño financiero con modelo estrella.

**Incluye:**
- Modelo de datos para BI (fact + dimensiones).
- Consultas SQL de extracción.
- Documentación DAX y guía de armado del dashboard.

### 3) ⚙️ [Pipeline ETL End-to-End](./proyecto_3_pipeline_etl)
**Objetivo:** integrar y transformar múltiples fuentes para producir un dataset maestro confiable.

**Incluye:**
- Fuentes raw sintéticas (clientes, órdenes, pagos).
- Proceso ETL con validaciones de calidad.
- Dataset procesado y reporte de data quality.

### 4) 🤖 [Análisis Predictivo de Churn (ML)](./proyecto_4_ml_predictivo)
**Objetivo:** predecir abandono de clientes y habilitar acciones de retención.

**Incluye:**
- Entrenamiento y evaluación de modelos (ROC-AUC).
- Artefacto de modelo serializado.
- Predicción batch y visualizaciones de desempeño.
